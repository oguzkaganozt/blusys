# Widget Gallery

Common stock widgets and where they fit best.

## Compact Control

- `button` - primary actions and confirmations
- `toggle` - on/off settings and simple state switches
- `slider` - ranged numeric control
- `gauge` - compact numeric feedback

## Operational Surface

- `card` - grouped content and summaries
- `list` - ordered options and drill-down navigation
- `tabs` - screen partitioning for denser apps
- `status_badge` - terse operational state

## Data And Diagnostics

- `key_value` - paired status fields
- `data_table` - structured operational rows
- `progress` - loading, OTA, and long-running work
- `dropdown` - compact choice selection

## Notes

- keep visuals driven by theme tokens
- use custom widgets only when the stock set no longer fits
- keep raw LVGL inside the widget implementation, not in app code

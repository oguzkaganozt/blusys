# Connected headless (minimal reference)

Small single-file example (`main/connected_headless_main.cpp`) showing connectivity + storage on the `blusys::app` path.

**For new connected products**, prefer the **[edge node quickstart](../../quickstart/edge_node/README.md)** archetype: it uses the standard `core/` / `integration/` split, full capability set, operational phases, and telemetry/OTA/diagnostics wiring aligned with the PRD.

This example remains useful as a minimal compile check and teaching slice, not as the main starting point.
